Git

  • Git is a free and open source version control system.
  • It is used to maintain different versions of a project so that we can easily switch to previous version if something unusual happens.
  • You can download git from here.

Github

  • Github is a code hosting plateform for version control and collaboration.
  • It lets developers to work together on a single project from anywhere.
  • You can use github from here.

Git and Github Configuration

To configure git and github you have to run following commands in the terminal

git config --global user.name "[firstname lastname]"

git config --global user.email "[valid-email]"

To fork the assignment repository go here and click on fork icon there.

Clone your forked repository with the command

git clone [repository address]

Create a new branch
git checkout -b [your-new-branch-name]

Add your files to the folder and commit them by

git add [filename]
git commit -m"[a short description]"

To push files you have to configure github repo with your computer.
Run following commands

ssh-keygen -t rsa -b 4096 -C "[your email]"
tail [path to id]

copy the key and add to your github account. Now run the following command.

git push origin master
