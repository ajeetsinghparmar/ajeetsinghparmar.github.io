Git

  • Git is a free and open source version control system.
  • It is used to maintain different versions of a project so that we can easily switch to previous version if something unusual happens.
  • You can download git from here.

Github

  • Github is a code hosting plateform for version control and collaboration.
  • It lets developers to work together on a single project from anywhere.
  • You can use github from here.

Git and Github Configuration

To configure git and github you have to run following commands in the terminal

git config --global user.name "[firstname lastname]" git config --global user.email "[valid-email]"
